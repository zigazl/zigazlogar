# Žiga Žlogar — Personal Website

Personal website of Žiga Žlogar, presenting work in IT projects, business analysis, and digitalization.

## About the project

This is the personal website of Žiga Žlogar.
The site presents my work in IT projects, business analysis, and digitalization.

It is built as a lightweight one-page website with a clean, Apple-inspired visual direction and is ready for deployment on Vercel.

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Running locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

This project is designed to be deployed on Vercel with no additional configuration required.

Production build:

```bash
npm run build
npm run start
```

## Project structure

- `app/` — Next.js App Router pages, layout, metadata, and global styles
- `components/` — reusable UI sections and shared components
- `data/` — site content and copy
- `public/` — static assets
